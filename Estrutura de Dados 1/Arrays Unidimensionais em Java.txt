Arrays Unidimensionais em Java:

Os Arrays Unidimensionais são os que possuem apenas um índice para acessar seu conteúdo eles são declarados da seguinte forma:
- Tipo de dado[] nome do array = new tipo-de-dado[quantidade]

Tipo de dado -> Pode ser qualquer tipo de variavel
Nome do array -> Um nome válido; As memas regras para nomes de variáveis 
Quantidade -> Quantidade de elementos que o array pode manipular

Exemplos:
int[] numeros = new int[10] Cria uma array com o nome numeros que contem 10 elementos do tipo int e seu indice varia de 0 a 9

Double[] preco = new Double[5] Cria um array com o nome preço que contém 5 elementos do tipo Double e seu indice varia de 0 a 4

char[] sexo = new char[2] Cria um array com o nome sexo que contem 2 elementos do tipo char e seu indice varia entre 0 e 1

boolean[] opcoes = new boolean [2] Cria um array com o nome opcoes que contem 2 elementos do tipo boolean e seu indice varia entre 0 e 1

String[] meses = new String[12] Cria um array com o nome meses que contem 12 elementos do tipo String e seu indice varia de 0 a 11
